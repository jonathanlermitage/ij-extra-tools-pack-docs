# Git Submodules Icon

If your project defines some git submodules, their containing folders icon should be customized. It works with nested submodules as well. You should see beautiful golden or green Git icons:

![](../../images/extra-icons/git-submodules.png){ width="840" }

![](../../images/extra-icons/git-submodules-settings.png){ width="840" }

> Submodules are detected on project loading (Extra Icons parses `.gitmodules` files only on project's load). If you check out submodules after project loading (in other words, if you create and populate submodule folders), you will have to re-open your project to see the new submodule icons.
